# EdTech Content Workflow
This project demonstrates how content is managed, reviewed, and improved in a digital learning environment.
## Workflow
1. Content submission by tutor  
2. Initial structure review  
3. Content quality review  
4. Feedback and revision  
5. Final approval  
## My Role
- Reviewing course structure and clarity  
- Improving learning flow  
- Coordinating with tutors  
- Ensuring consistency across lessons  
## Tools Used
- Notion  
- Slack  
- Google Docs  
## Goal
To build a clear, scalable system for managing educational content and improving learner experience.
